There are three different types of interactions reoresented by a directed edge(u,v,t)
1. User u answered user v's question at time t
2. User u commented on user v's question at time t
3. User u commented on user v's answer at time t


Source(citation)
Ashwin Paranjape, Austin R. Benson, and Jure Leskovec. "Motifs in Temporal Networks." In Proceedings of the Tenth ACM International Conference on Web Search and Data Mining, 2017.
